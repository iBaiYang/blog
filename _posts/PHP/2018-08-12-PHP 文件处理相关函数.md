#### 解说

##### file_exists

file_exists('文件地址')  // 检查指定文件是否存在

如：
```
$addr = '/data/log/swoole/swoole_channel_socket.log';
$pid = file_exists($addr) ? file_get_contents($addr) : null;
```

##### file_get_contents

file_get_contents('文件地址')  // 将文件读入字符串

##### file_put_contents

file_get_contents('文件地址', '内容字符串', '如何写入文件')  // 将字符串写入文件
