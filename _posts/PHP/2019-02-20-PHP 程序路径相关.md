### 正文

```
getcwd()  // 显示是 在哪个文件里调用此文件 的目录

__DIR__   // 当前内容写在哪个文件就显示这个文件目录

__FILE__   // 当前内容写在哪个文件就显示这个文件目录+文件名
```

另外配合用的比较频繁的还有 
```
basename()  // 函数返回路径中的文件名部分

dirname(path)  // 返回路径中的目录部分

pathinfo()  // 返回文件路径的信息

realpath()  // 返回规范化的绝对路径名
```
