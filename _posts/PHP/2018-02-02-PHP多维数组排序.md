> 在php中，对于多维数组排序，有可以直接使用的函数，不需要自己重写。
> 
> array_multisort()，可以看一下文档，php.net上的。
> w3school和菜鸟的就不要看了，言之不详。

举个例子：
<pre><code>
$data = [
    ['id' => 13, 'name' => 'sam'],
    ['id' => 8, 'name' => 'tom'],
    ['id' => 46, 'name' => 'addy'],
];
$orderArr = array();
foreach ( $data as $val )
{
    $orderArr[] = $val['id'];
}
array_multisort($orderArr, SORT_ASC, SORT_NUMERIC, $data); 
</code></pre>
