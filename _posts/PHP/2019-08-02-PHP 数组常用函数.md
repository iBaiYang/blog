### 正文

#### array_map

```
$str = 'abc';

$arr = [1,2,3];

$arr = array_map(function($v) use($str) {
    $v *= $v;
    $v .=  $str;
    return $v;
}, $arr);

print_r($arr);

输出：
Array
(
    [0] => 1abc
    [1] => 4abc
    [2] => 9abc
)
```

#### compact

```
$city  = "San Francisco";
$state = "CA";
$event = "SIGGRAPH";

$location_vars = array("city", "state");

$result = compact("event", "nothing_here", $location_vars);
print_r($result);

输出：
Array
(
    [event] => SIGGRAPH
    [city] => San Francisco
    [state] => CA
)
```
