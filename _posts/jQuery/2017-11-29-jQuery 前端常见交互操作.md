如果是键盘回车事件呢？

```javascript
$(document).on('keydown','.content-filter #mobile-search',function (e) {
    var theEvent = e || window.event;
    var code = theEvent.keyCode || theEvent.which || theEvent.charCode;
    if ( code == 13 ) {
        e.preventDefault();
        console.log('get');
    }
}); 
```
